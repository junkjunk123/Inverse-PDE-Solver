This project models a differential PDE-solver that solves inverse problems, specifically in the field of microfluidics. The project experiments with the Finite Element Method (FEM) for solving partial differential equations, working under Sobolev Space to find discrete solutions. Experimenting with the JAX framework, the project strives to create a JAX-FEM PDE-solver that allows for differentiation with respect to parameters. This enables the project to solve inverse problems, optimizing over parameter space to produce a desired microfluidic laminar flow. Starting with simpler two-dimensional partial differential equations, the project works towards creating a stable model of Stokes flow, beginning in two-dimensions.
